## Application Overview

XtraFinder serves users wanting Finder improvements without switching to entirely different file managers. The enhancements integrate naturally with existing Finder workflows.

Show hidden files toggle reveals system files when needed without terminal commands. Quick access serves developers and advanced users who frequently need hidden file visibility.

Folders on top sorting groups directories before files in name-sorted views. The arrangement matches behavior users expect from other file managers.

Middle-click opens folders in new tabs for rapid multi-location browsing. Mouse-centric navigation becomes more efficient.

Colorful labels enhance file visibility through folder coloring beyond standard Finder tags. Visual organization helps identify important items quickly.
